Part 1 — Is this workflow worth automating?

Score one point for every “yes.” Pick your most painful workflow and run it through:

  • 01High volume. Does this happen dozens or hundreds of times a week?
  • 02Repetitive. Is it roughly the same steps every time?
  • 03Rule-based core. Could you write down the decision logic, even if there are exceptions?
  • 04Costs real hours. Is a person (or several) spending meaningful time on it weekly?
  • 05Measurable. Can you count the outcome — tickets, leads, orders, hours?
  • 06Bottleneck. Does it slow down other work or hurt response times when it backs up?

Part 2 — Are you ready to build?

  • 07Accessible data. Does the info the workflow needs live somewhere reachable (a CRM, helpdesk, database, API)?
  • 08Nameable systems. Can you list the 2–4 tools it would need to touch?
  • 09A baseline. Do you know the current cost/time so you can prove improvement?
  • 10An owner. Is there one person who understands the workflow and can answer questions?
  • 11Tolerance for a pilot. Can you start with one workflow instead of boiling the ocean?
  • 12Guardrails clarity. Do you know which actions need human approval vs. can run automatically?

9–12 yes: Strong candidate for a custom AI agent. Start a fixed-scope pilot now.

5–8 yes: Promising — a chatbot or a scoped agent pilot likely fits; tighten the data and baseline first.

Under 5 yes: Not yet. Either pick a higher-volume workflow or fix data access before automating.
